# 삶의 가치를 더하는 개발자 박건희

아이디어를 현실로, 세상과 소통하는 코드를 작성합니다.

## 🛠 기술 스택
### Back-end
* Spring Boot 3, Java 17
* Python, YOLOv5/8
* MyBatis, JPA
* MySQL 8 (설계·인덱싱·쿼리 최적화 경험)
* Redis(ElastiCache), JWT 인증/권한 구조
### Front-end
* Node.js, Vue 3, Vite, Pinia
* 커스텀 테이블/모달 등 컴포넌트화
* 대시보드 제작 (데이터 분석)
### DevOps / Infra
* AWS EC2, RDS, Redis
* Nginx + systemd 배포
* VPC·보안그룹 설계, NAT 비용 최적화
* 운영 중 핫픽스 경험
